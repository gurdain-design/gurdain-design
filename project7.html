Parry J / Website

Parry J is a singer from the UK. The purpose of the website was to direct listners to Parry J's discography and to stream the latest release, as well as directing to Parry J's active social media accounts. The home page features a looping video from Parry J's music video 'Stay With Me' shot and directed by Jaber. The website is built upon HTML & CSS. It also includes JavaScript for the mobile navigation menu. You can visit the website here - parryj.com
