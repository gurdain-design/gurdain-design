Celestial / Website

Celestial is a producer/artist from the UK. The website is intended to group together all releases made by Celestial and direct listners to the available mediums to play the releases. The homepage contains a looped video featuring stock footage that was recoloured to fit a purple colour scheme the is prevelent in Celestial's visuals. It showcases the 'Wander EP' and provides a clickable link when the various streaming icons appear. The website is built upon HTML & CSS. It also includes JavaScript for the mobile navigation menu. You can visit the website here - mindofcelestial.com
